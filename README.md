A Python-based desktop music player built using Tkinter and Pygame.

## Features
- Load MP3 and WAV files
- Playlist support

- Next and Previous controls
- Neon-themed GUI

## Technologies Used
- Python
- Tkinter
- Pygame

## How to Run
1. Install pygame:
   pip install pygame
2. Run the program:
   python music_player.py
